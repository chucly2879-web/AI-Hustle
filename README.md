# AI Side Hustle Explorer 🚀

**AI Side Hustle Explorer** là một ứng dụng web hiện đại giúp người dùng khám phá và xây dựng các nguồn thu nhập trực tuyến bằng sức mạnh của Trí tuệ nhân tạo (AI).

## 🌟 Tính năng chính
- **Trình tạo ý tưởng cá nhân hóa:** Sử dụng Gemini 2.0 Flash để thiết kế lộ trình kiếm tiền dựa trên kỹ năng riêng của bạn.
- **Thư viện Prompt:** Tổng hợp các câu lệnh mẫu tối ưu cho viết blog, đăng sản phẩm Shopee/Lazada và Marketing.
- **Danh mục nghề nghiệp AI:** Gợi ý các lĩnh vực hot nhất như Content Writing, AI Design, E-commerce Optimization.
- **Blog chuyên sâu:** Chia sẻ kinh nghiệm thực chiến về việc ứng dụng AI trong công việc Freelance.

## 🛠 Công nghệ sử dụng
- **Frontend:** React 19, Vite, TypeScript.
- **Styling:** Tailwind CSS 4.
- **Animations:** Framer Motion.
- **Icons:** Lucide React.
- **AI Engine:** Google Gemini API (@google/genai).

## 🚀 Bắt đầu
1. Clone dự án: `git clone https://github.com/chucly2879-web/aihustle.git`
2. Cài đặt dependencies: `npm install`
3. Chạy môi trường dev: `npm run dev`

## 📝 Giấy phép
Dự án này được tạo ra nhằm mục đích truyền cảm hứng cho cộng đồng Freelancer Việt Nam.
